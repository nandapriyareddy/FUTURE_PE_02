# FUTURE_PE_02
# UGC Ad Content Pack – VelvetVows Celebrations

## Business Chosen

VelvetVows Celebrations is an event planning and decoration business that specializes in weddings, engagements, birthdays, anniversaries, baby showers, corporate events, and custom-themed celebrations. The focus is on creating elegant, emotional, and memorable experiences through professional décor and event styling.

## Prompt Logic

The prompts were designed to generate high-converting UGC-style ad content that feels authentic, emotional, and relatable. Each prompt follows a structured approach using hooks, storytelling (Problem → Solution → Result), and strong call-to-actions. The system ensures multiple variations of hooks and scripts to support scalable content creation for different marketing campaigns.

The goal was to simulate real customer experiences to increase trust and engagement on platforms like Instagram, Facebook, and YouTube Shorts.

## Tool Used

ChatGPT was used to generate all UGC ad content, including hooks, scripts, and captions. The tool helped maintain consistent tone, emotional storytelling, and structured ad formats suitable for social media marketing.

## Repository Contents

* prompts.txt → Structured prompts used to generate UGC content
* hooks-output.txt → Collection of attention-grabbing hooks
* ugc-scripts-output.txt → Full UGC ad scripts
* cta-output.txt → Call-to-action statements
* captions-output.txt → Social media captions
